# v1.16.0-rc.8 — Enricher bundle (threatintel-domain + geoip-asn + codesigning-anomaly)

Three §13.8 enrichers ship as one RC. Each is paired with an idempotency test per Pass 2026-C.

## What landed

**`com.maccrab.enricher.threatintel-domain`** — annotates the subject with `threatintel.domain` + `threatintel.is_known_malicious` + `threatintel.is_known_phishing`. Built-in static IOC set in this RC; live `MacCrabCore.ThreatIntelFeed.cachedIOCs()` integration deferred. Pure function on subject — Pass 2026-C invariant trivially provable.

**`com.maccrab.enricher.geoip-asn`** — annotates the subject's payload with `geoip.range_token` for any IPv4 found. Recognizes loopback / private class A/B/C / CGNAT / link-local / public unknown. Live MMDB binding deferred until operator opts into MaxMind license; the API shape is stable.

**`com.maccrab.enricher.codesigning-anomaly`** — flags surprising codesign postures: unsigned binary in `/usr/bin` or `/System/`; Developer-ID signed binary in `/System/`; ad-hoc signed binary outside `/Users/` or `/tmp/`; Developer-ID unnotarized in a system area. Returns the anomaly flag set as `codesigning_anomaly.flags`.
