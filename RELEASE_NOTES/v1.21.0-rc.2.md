# MacCrab v1.21.0-rc.2

Builds on rc.1 with catalog and Overview improvements.

## Forensics catalog

- An installed plugin now always has a working action: **Update to vX** when the catalog has a newer version, otherwise **Reinstall** (a verified re-install of the current version), alongside the "up to date" status and **Remove**. Fixes the case where the upgrade/install button did nothing or errored.

## Overview dashboard

- **Reset to default** now confirms before discarding a customized layout.
- Three new widgets (all hideable, movable, and resizable like the rest):
  - **Protection Coverage** — live sensor health ("N of M sensors healthy") so a silently-stopped collector is visible at a glance.
  - **Top Firing Rules** — the noisiest detections by fire count, with last-fired time.
  - **Crabby** — the MacCrab mascot scuttles across the card and reacts to your live posture (calm when quiet, watchful on alerts, claws-up on a critical campaign, happy when all-clear). Honors Reduce Motion.

## rc.1 (carried forward)

- Overview resize fills rows neatly (no ragged gaps).
- Broad allowlists can't silence active C2 / credential-theft alerts.
- Forensic plugin platform reachable from the MCP server (list cases, search catalog, run scanners, install/update/uninstall/pin under the response capability).
- Valid-JSON event storage fix; SIGHUP rule-reload hardening; trust-consent and accessibility polish.
