# MacCrab v1.7.10 — About-page version fix + v1.7.9 rollout

UX hot-fix on top of v1.7.9. Two things ship together:

## Fixed — About page version label

`Settings → About` was reading "v1.3.4" — hardcoded ~20 releases ago
and never updated. Replaced with a `Bundle.main.infoDictionary` read
so every future release is correctly self-reported without a manual
code edit.

## v1.7.9 rollout

v1.7.9 had its Sparkle appcast and Homebrew tap intentionally held
pending memory-fix verification on a second test machine.
Verification confirmed the outer-loop autoreleasepool wraps fixed
the 1.31 GB Foundation-autorelease accumulation. v1.7.10 bundles
the About-page fix and rolls v1.7.9's full content out to all
channels in one shipping action.

Everything in v1.7.9's release notes also applies here — see
[v1.7.9.md](./v1.7.9.md) for the full memory diagnostic story,
the new Pass 9 + Pass 10 audits, the 3 new detection rules
(crypto wallet + XPC service replacement + unsigned Network
Extension), the heartbeat counter fix, the brew/Sparkle drift
detection, the metrics export schema-2 changes, and the UX
cleanup (zombie banner removed, sidebar reworked).

## How to install

### Sparkle auto-update (now active)

Open MacCrab.app. Update banner appears within minutes of appcast
publish. Click "Install Update".

### Homebrew (now active)

```bash
brew upgrade --cask maccrab
```

### Fresh DMG

[github.com/peterhanily/maccrab/releases/v1.7.10](https://github.com/peterhanily/maccrab/releases/tag/v1.7.10)

## What you don't need to do

- **No reboot.** sysextd hot-swaps the binary on update.
- **No extension re-approval.** Same team-id + bundle-id.
- **No data migration.** Storage layer unchanged from v1.7.6.
