# v1.16.0-rc.13 — codesigning-graph + Office document analyzer

**`com.maccrab.forensics.codesigning-graph`** — walks `/Applications`, `~/Applications`, `/usr/local/bin`, `/opt/homebrew/bin`, `/opt/homebrew/sbin`, `/Library/PrivilegedHelperTools`. Resolves `.app` bundles to their `CFBundleExecutable`. Emits one `codesigning.binary` artifact per binary with `signer_type` + `team_id` + `is_notarized`. Pairs with launchd-lite for fleet-wide trust posture: every binary either Apple-signed, Developer-ID from a known team, or unsigned/unknown.

**`com.maccrab.forensics.office-document-analyzer`** — OPC zip + XML parser for `.docx` / `.docm` / `.xlsx` / `.xlsm` / `.pptx` / `.pptm`. Extracts `creator` / `last_modified_by` / `created_iso` / `modified_iso` / `title` from `docProps/core.xml`. Flags macro-bearing documents (presence of `vbaProject.bin`).

Internal RC. 22 tags total.
