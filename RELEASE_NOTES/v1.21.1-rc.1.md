# MacCrab v1.21.1-rc.1

A small fix to the menu bar icon.

## Fixes

- **Menu bar icon spacing.** The menu bar now always shows a single,
  normally-spaced crab. Previously a live alert or a degraded-protection
  state prepended a second glyph (a colored severity dot, or a warning
  triangle), which macOS rendered as a wider, double-width icon with an
  awkward gap next to the crab. Protection health and alert state are still
  surfaced in full in the dashboard, in system notifications, and to
  VoiceOver.

## Notes

This is a release candidate for on-device verification; it is not published
to the auto-update channel. It carries no detection-engine changes from
v1.21.0 — the rules, sequences, and graph rules are unchanged.
