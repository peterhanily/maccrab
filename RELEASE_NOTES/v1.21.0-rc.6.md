# MacCrab v1.21.0-rc.6

## Overview dashboard — Crabby

- Crabby's sprite is now the **rave.maccrab.com pixel crab**, with a different **face per mood** (thriving, dozing, uneasy, alarmed — and smitten when you pet him).
- He's **interactive**: **Pet** him (a happy wiggle), **Acknowledge** an alarm (he settles back down), or **Investigate** (jump to the campaign).
- **Acknowledging an alarm** settles Crabby to calm until a worse event happens — no more permanent "alarmed".
