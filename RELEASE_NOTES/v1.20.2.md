# MacCrab v1.20.2

A small stability and hardening release.

## Dashboard

- **Customizing the Overview is clearer and smoother.** It's now obvious you can
  rearrange your dashboard: each card shows a drag handle and a grab cursor while
  customizing, with a hint explaining how to drag, resize, and hide panels — and
  reordering no longer jumps as you drag.

## Security

- **Sandbox hardening.** The forensic plugin sandbox's file-descriptor broker now
  bounds-checks an incoming control message, so a malformed message from a
  compromised plugin can't read past its buffer. The plugin sandbox already
  refuses untrusted code; this closes a defense-in-depth gap on the broker.

## Tooling

- **Out-of-band rule updates — sender side.** Added operator tooling to author and
  Ed25519-sign rule-update manifests for the signed, app-decoupled rule channel
  (`scripts/build-rules-manifest.sh`). Pushed rules remain anti-rollback,
  version-floored, additive-only, and detection-only.

## Compatibility

No configuration changes are required. macOS 13 (Ventura) or newer.
