# MacCrab v1.20.0

A dashboard-customization and rule-delivery release: the Overview is now a
customizable dashboard whose layout you control and that persists across
updates, "Run a scan" is simpler, and detection rules can ship out-of-band from
a signed channel without an app update.

## Customizable Overview dashboard

- **Make it yours.** Show or hide any panel, drag panels to rearrange them, and
  resize each panel to a 1–4 column width. Enter "Customize", arrange the
  dashboard the way you work, and click Done.
- **Your layout sticks.** The arrangement is saved and persists across app
  updates. It's also resilient to a corrupted preferences file — a bad layout
  recovers to sensible defaults instead of crashing or wiping your setup.
- **Safety stays visible.** The protection-status banner is pinned and always
  shown, so you can never accidentally hide whether you're protected.
- The Event Rate and Threat Intel panels are back.

## Forensics

- **Simpler "Run a scan".** Two clear lists — Built-in scanners and Installed
  plugins — each collapsible and expanded by default. The kit picker was removed.
- **Provenance at a glance.** Each forensic plugin is labelled Built-in, Store,
  or Sideloaded.

## Out-of-band rule updates

- **Detection rules can update separately from the app.** New rules can ship from
  a signed channel without an app update or restart (`maccrabctl rules update`).
- **Trust by construction.** Pushed rules are Ed25519 signature-verified and
  anti-rollback protected, and are strictly detection-only — they raise alerts
  but never trigger a response action and never override a built-in rule.

## Privacy & hardening

- Cloud LLM backends gain a strict no-leak mode that refuses a request if any
  sensitive content survives sanitization.
- A false-positive-rate benchmark harness lets the per-rule FP rate be measured
  on a real machine.

## Compatibility

No configuration changes are required. macOS 13 (Ventura) or newer.
