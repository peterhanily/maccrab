# MacCrab v1.21.0-rc.3

Polishes the Overview changes from rc.2.

## Overview dashboard

- **No more grid gaps.** The customizable grid now densely packs widgets — resizing a panel (or resizing the window) no longer leaves a ragged hole; a later widget slides up to fill the space.
- **Crabby is calmer.** He scuttles more slowly and only reacts to *active campaigns* — old or benign rule firings (and a merely-imperfect security score) no longer keep him in a panic. Claws-up only for a genuinely active critical campaign.
- **Top Firing Rules shows real names.** The card now displays human rule titles instead of raw rule IDs.

## Carried forward from rc.1 / rc.2

- Installed catalog plugins always have a working action (Update / Reinstall / Remove) + "up to date" status.
- Reset-to-default for the Overview now confirms first.
- New widgets: Protection Coverage, Top Firing Rules, Crabby.
- Forensics CLI↔MCP parity; broad allowlists can't silence active C2 / credential-theft alerts; valid-JSON event storage; trust-consent + accessibility polish.
