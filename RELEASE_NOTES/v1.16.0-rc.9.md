# v1.16.0-rc.9 — Four §13.7 file analyzers (plist + mobileconfig + shortcuts + image-metadata)

Four file analyzers bundled. All four are property-list / ImageIO based — no subprocess dependencies; pure parser surfaces.

## What landed

**`com.maccrab.forensics.plist-analyzer`** — emits `plist.analysis` per .plist: format (binary / xml / openstep) + top-level keys + size + sha256.

**`com.maccrab.forensics.mobileconfig-inspector`** — emits `mobileconfig.analysis` per profile: CMS-signed flag + payload types + `has_certificate_payload` / `has_dns_payload` / `has_vpn_payload`. Risk surface: MDM profiles can install root certs, change DNS, deploy VPN.

**`com.maccrab.forensics.shortcuts-analyzer`** — emits `shortcuts.shortcut` per `.shortcut` / `.wflow` file found under `~/Library/Shortcuts` + the iCloud Shortcuts library. Catalogs name + size + mtime + sha256. Action-graph parse deferred (proprietary binary keying).

**`com.maccrab.forensics.image-metadata`** — emits `image.metadata` via `ImageIO`: width / height / DPI / color model + `has_gps` / `has_exif` / `has_iptc` flags. Risk surface: tracking pixels in downloaded images, GPS-tagged screenshots, document leakage via embedded thumbnails.
