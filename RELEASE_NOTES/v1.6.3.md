## MacCrab v1.6.3 — The crab is back

Hotfix in response to immediate user feedback on v1.6.2: the menu-bar icon
is once again the 🦀 emoji, not the SF Symbol shield.

v1.6.2 swapped in a template-rendered `shield.lefthalf.filled` for cleaner
light/dark menu-bar styling — but the crab is the brand. Plenty of popular
Mac apps (1Password, Discord, Notion) use colorful menu-bar icons for
exactly this reason: brand recognition wins over strict template rendering.

### What's restored

- Healthy state: 🦀
- Degraded state (zero rules loaded, stale heartbeat, storage errors): ⚠️🦀
- Live critical alert flash: 🔴🦀 (10 s)
- Live high-severity alert flash: 🟠🦀 (10 s)
- Menu header: "🦀 MacCrab"

### What's kept from v1.6.2

- Accessibility label on the menu-bar item ("MacCrab" / "MacCrab —
  protection degraded" / "MacCrab — critical severity alert") so VoiceOver
  still gets a meaningful announcement instead of "crab emoji."
- Dedup on title assignments (only updates when the displayed title
  actually changes) — costs nothing, avoids needless redraws.
- Everything else from v1.6.2 (site-matching MacCrabTheme, About panel
  with maccrab.com link, Help menu, Info.plist completeness, performance
  fixes, Reduce Motion respect) is untouched.
