# v1.16.0-rc.14 — three opt-in content-class collectors

Three companion plugins to the metadata-first collectors from earlier RCs. **All three emit content-class artifacts**; Pass 2026-D rejects at INSERT for plaintext cases.

- **`com.maccrab.forensics.imessage-bodies`** — opt-in companion to `imessage-metadata`. Emits `imessage.message_body` artifacts with full text. Privacy class **content**.
- **`com.maccrab.forensics.mail-bodies`** — opt-in companion to `mail`. Walks `.emlx` files for From/To/Subject + 32 KB body preview. Full MIME multipart walk + attachment extraction deferred. Privacy class **personalComms**.
- **`com.maccrab.forensics.safari-deep`** — opt-in companion to `safari-lite`. Discovers LocalStorage + IndexedDB databases per origin (no value bodies in this RC; just origin + size). Cookies.binarycookies parser deferred. Privacy class **content**.
