# MacCrab v1.21.0-rc.4

## Overview dashboard

- **Mixed-height widgets no longer leave gaps.** The customizable grid now uses a masonry layout: each widget keeps its natural height and tucks into the shortest available spot, so a short tile next to a tall card no longer leaves dead space. Combined with the rc.3 packing, the dashboard stays gap-free at any window size.

## Carried forward (rc.1 → rc.3)

- Installed catalog plugins always have a working action (Update / Reinstall / Remove) + "up to date" status.
- Crabby is slower and only reacts to active campaigns; Top Firing Rules shows real rule names.
- Reset-to-default confirms first; new widgets: Protection Coverage, Top Firing Rules, Crabby.
- Forensics CLI↔MCP parity; broad allowlists can't silence active C2 / credential-theft alerts; valid-JSON event storage; trust-consent + accessibility polish.
