# v1.16.0-rc.10 — DMG/PKG + archive analyzers

**`com.maccrab.forensics.dmg-pkg-analyzer`** — operator-supplied .dmg / .pkg analysis. Shells out to `hdiutil imageinfo` (for DMG) / `pkgutil --check-signature` (for PKG). Wraps with `CodeSigningCache` for the container's signer_type + team_id. Catches repackaged-installer supply-chain artifacts. Scans `~/Downloads` by default until per-invocation inputs land.

**`com.maccrab.forensics.archive-walker`** — listing-only walker for `.zip` / `.tar` / `.tar.gz` / `.tgz`. Shells to `unzip -l` / `tar -tvf`. Emits archive entry count + filename preview (capped 50). No extraction.

Internal RC.
