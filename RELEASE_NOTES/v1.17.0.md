# MacCrab v1.17.0 — Forensics workspace

v1.17.0 is the first public release since v1.12.9. It introduces the
**Forensics workspace**: on-demand, on-device evidence collection you
run yourself, with results that never leave your Mac.

## Why the version jumps from 1.12.9 to 1.17.0

The last public release was v1.12.9. Versions v1.13.0 through v1.16.0
were internal development milestones only — never shipped to the public
— and are where the new Forensics platform was built up incrementally.
v1.17.0 is the first public release to include that finished, user-facing
Forensics workspace (along with the fixes below), so the public version
moves straight from 1.12.9 to 1.17.0.

## New

- **Forensics workspace.** A four-tab workspace — Run a scan, Past
  scans, Findings, and Catalog. Pick a curated scan kit (incident
  response, phishing triage, comms forensics, supply-chain audit,
  browser activity, Mac data overview, and more), run it, and review
  the results.
- **Evidence viewers.** Scan results render as sortable tables,
  day-grouped timelines, message transcripts, and charts — chosen to
  fit the artifact — instead of raw JSON.
- **Findings feed.** An actionable summary across your scans, with
  severity at a glance.
- **Plugin catalog (preview).** A preview of the upcoming signed
  plugin catalog. Your installed scan kits work today; in-app browsing
  and install arrive in a later release.
- **Export.** Save any scan's evidence as CSV or JSON.
- **Manage past scans.** Set an auto-delete age for old scans, see each
  scan's disk usage, select and bulk-delete scans, and permanently
  delete a scan when you're done with it.
- **Full Disk Access guidance.** When a scan needs access, MacCrab
  shows a clear prompt with a direct link to the right System Settings
  pane.
- **Command line.** `maccrabctl` gains `case` and `plugin` subcommands
  for running and managing forensic scans from the terminal.

Scan evidence is stored in per-case encrypted databases under
`~/Library/Application Support/MacCrab/Cases/`.

## Fixes & improvements

- **Threat-intel feeds update reliably again.** A line-ending bug made
  the abuse.ch feeds (URLhaus, MalwareBazaar, Feodo Tracker) parse no
  records, so indicator counts never refreshed. Feeds now fetch and
  parse correctly, report honestly when a fetch fails or is stale, and
  the Intelligence workspace surfaces recent indicator matches.
- **Clearer critical notifications.** Critical alerts now always notify
  at their true critical severity. The confusing "Allow critical
  notifications" toggle has been removed (older config files still
  load).
- The detection engine and dashboard also pick up refinements from this
  development cycle.

## Upgrade

- **Homebrew:** `brew upgrade --cask maccrab`
- **Sparkle:** accept the in-app update prompt and relaunch when asked.
- **Manual:** download the DMG from the release page and replace
  MacCrab.app.

Universal (Apple silicon + Intel), Developer ID signed and notarized.
Minimum macOS 13.
