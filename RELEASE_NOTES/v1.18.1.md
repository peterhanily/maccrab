# MacCrab v1.18.1

A dashboard quality release: the app is now genuinely usable in 14 languages, alert triage and rule management get faster and cleaner, the dashboard stops doing redundant work while you move the mouse, and VoiceOver users get real semantics across the main surfaces.

## Localization

- **13 languages go from ~35% to 93–97% translated** — German, Spanish, French, Italian, Japanese, Korean, Dutch, Polish, Brazilian Portuguese, Russian, Swedish, Simplified and Traditional Chinese. Detection content (rule names, MITRE techniques) intentionally stays English — the international SOC vocabulary — while the entire UI layer translates.
- **Section headers that ignored your language are fixed.** A class of dashboard headers ("What happened", "Actions", "Description", …) was hardcoded English regardless of locale; they now localize like everything else.
- Translations are marked for native-speaker review, and a new `TRANSLATION.md` documents the glossary and rules contributors should follow.

## Alerts & detection rules

- **Cleaner alert actions.** The alert inspector's actions are a proper action bar — primary action on top, secondary actions in a compact row — and the alert notification popover got a severity-tinted refresh.
- **Change rule settings without touching YAML.** The rules table's status dot is now a click-to-toggle enable/disable, and the severity chip opens a menu to override severity (Critical → Info, or back to Default) directly in the row — for built-in detections and Sigma rules alike. The rule inspector gains the same controls. Multi-step sequence/graph rules remain read-only, with an explanation.
- **History tab fix:** a suppressed recent alert could appear twice in the list; it now appears once.

## Performance

- **Hovering over tables no longer re-sorts them.** Every shared dashboard table (Alerts, Rules, Intelligence, System) re-filtered and re-sorted up to 1,000 rows on every pointer movement; row hover is now isolated and the sort/filter result is cached until something actually changes.
- **Fixed a memory leak class in the Events aggregate view.** Parking the dashboard on a "Last 7 days" or "All time" range slowly accumulated layout objects; the aggregate table now rebinds only when its data changes.

## Accessibility

- **Command palette:** VoiceOver now tracks the keyboard-selected result, announces selection moves and result counts, and recognizes section headers.
- **Charts speak.** The event histograms and forensic charts expose a one-sentence summary (total, bins, peak) instead of being silent images.
- **Sidebar:** group headers are real headers, the resize handle is adjustable via VoiceOver, and labels read naturally instead of spelling out "⌘N".
- New `docs/ACCESSIBILITY.md` tracks the WCAG-AA checklist.

## Reliability & build hygiene

- The release pipeline now hard-fails if the localization bundle would ship incomplete, records the exact build toolchain in `release.json`, and pins release builds to Xcode 26.x with guards against two known Xcode-27 source-compatibility hazards — groundwork for a safe toolchain migration later this year.
