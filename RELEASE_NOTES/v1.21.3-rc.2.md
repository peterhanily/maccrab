# MacCrab v1.21.3-rc.2

Catalog fixes on top of rc.1.

## Fixes

- **Installed plugins are recognised in the catalog.** An already-installed
  plugin now shows "Installed · up to date" (or "· update available" with an
  Update button) instead of a fresh Install button.
- **Run a plugin right after installing it** — the installed-plugin actions now
  include a Run button that runs it on this Mac immediately.

## Carried from rc.1

Built-in scanners list immediately; "Run on this Mac" no longer jars the window;
plainer wording for skipped plugins; and sandboxed community plugins can read
their declared files through the mediated broker (MacCrabPluginKit).

## Notes

Release candidate for on-device validation; not published to the auto-update
channel. No detection-engine rule changes.
