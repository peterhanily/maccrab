# v1.16.0-rc.11 — Document analyzer (PDF)

**`com.maccrab.forensics.document-analyzer`** — PDF dissection via PDFKit. Emits `document.analysis` per `.pdf` in `~/Downloads` (default scan target):

- `page_count` + author / producer / creator / creation / modification dates from `PDFDocument.documentAttributes`.
- `has_javascript` — substring scan for `/JavaScript` or `/JS` markers in the raw PDF.
- `has_embedded_file` — substring scan for `/EmbeddedFile` (the typical exfil-via-PDF carrier).

Office formats (.docx / .xlsx / .pptx — OPC zip + XML) deferred to a follow-up sub-slice; their content extraction is heavier engineering.
