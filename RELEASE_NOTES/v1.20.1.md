# MacCrab v1.20.1

A maintenance release. There are **no user-facing changes** from v1.20.0 — the
customizable Overview dashboard, the simplified "Run a scan", and the signed
out-of-band rule-update channel all behave exactly as in v1.20.0.

This release re-cuts the build through the complete automated release pipeline
with every pre-flight safeguard green.

## Internal

- Realigned the release pipeline's pre-flight safeguards so the full automated
  release runs end-to-end: the Settings → daemon binding classification now
  covers the opt-in network-enrichment flags, and the SQLite schema-migrator's
  idempotent re-run branch is explicitly marked.
- Made the release publisher resilient to a stale site/tap credential so the
  Sparkle appcast and release manifest always publish from the live credential.

## Compatibility

No configuration changes are required. macOS 13 (Ventura) or newer. If you are
already on v1.20.0, this update is optional — it changes no app behavior.
